## 4. Key Recommendations

1.  **Introduce Secondary Unconverged Signals (ROI: High):**
    Allow velocity spikes $> 5.0	ext{x}$ with high mention volume (e.g. today's mentions $\ge 5$) to generate secondary signals. These can be validated using technical breakout filters (volume/price action) instead of requiring a corporate calendar event.
    
2.  **Sector Theme Mapping (ROI: Medium):**
    Implement a constituents mapping for themes. When a sector theme (like `Digital Infrastructure`) experiences a Z-score spike, generate relative-value long signals on the top 3 constituent stocks of that sector.
